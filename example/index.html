Usage

Include or require Quill and QuillDropzone.

var editor = new Quill('.editor', { modules: { 'dropzone': { container: '.dropzone-selector', // Dropzone settings. config: { url: '/upload/file', headers: { 'Accept': 'application/json', 'Content-Type': 'application/hal+json' }, init: function() { // Example extension. Quill.editors[0].editor.quill.onModuleLoad('dropzone', (dropzone) => { this.on('success', function(file, xhr) { // Handle response from server. let location = xhr.location ? xhr.location : ''; // Attach insert buttons under each preview. dropzone.actions(file.previewElement, window.location.origin + location); }) }); } } } } }

Methods

Insert
Insert media asset at current (cursor) range.

Actions
Provide insert button on Dropzone preview container.
